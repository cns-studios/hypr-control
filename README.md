# Hypr Control

A CLI Settings app running on **NodeJS**

## Setup

Be sure to have `NodeJS` installed on your Arch Hyprland System


**Clone this repo** and enter it
```bash
git clone https://github.com/cns-studios/hypr-control.git
cd hypr-control
```

**Install dependencies**
```bash
npm i 
```

**Make script executable**
```bash
chmod +x hypr-control.js
```

**Run it** via

- hypr-control
- `node hypr-control`
- `npm start`
